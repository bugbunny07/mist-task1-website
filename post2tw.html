A Day in My Life

Posted on September 25, 2025

Today was one of those days that reminded me why I love learning something new every day. I spent most of my time experimenting with HTML and CSS to create my very own personal website.

I started the day by revising how to structure a webpage using <div> containers and how CSS can control spacing, colors, and fonts. I also discovered how satisfying it is to use Tailwind CSS to build clean designs quickly!

Later, I took a walk outside and realized how similar web development feels to art — it’s all about creativity and structure. Each line of code feels like a brushstroke that shapes something beautiful.

I ended the day feeling productive and inspired to keep coding. Tomorrow, I want to try adding images and maybe a small gallery section!
